Saiba qual o seu tipo de agendamento

Medicamentos do Município

São medicamentos com receitas simples, como antidepressivos, medicamentos controlados, antibióticos, medicamentos para pressão, coração, e doenças similares.

Medicamentos do Estado

São medicamentos geralmente de auto custo e para doenças mais raras, eles são fornecidos pelo estado, porém esta opção de agendamento é para casos onde o paciente já faz uso do medicamento, o seja já realizou todo processo inicial do pedido do mesmo.

Atendimento de Processo

Esta opção como vimos a cima serve para efetuar o agendamentopara o início do pedido para os medicamentos do estado, sempre que o paciente necessitar de algum tipo de medicamento de auto custo (medicamento do estado) do qual ainda não faz uso, ou ainda não usuflui do benefício essa é a opção que deve ser escolhida
